# Exquisite Portfolio — Christella

A high-end, typography-centric professional portfolio built with **Next.js 16** and **React 19**. This project showcases a minimalist, premium aesthetic inspired by modern design trends (Agex), featuring smooth animations, interactive components, and a perfectly aligned grid system.

![Portfolio Preview](/public/meeh.jpeg)

## ✨ Features

- **Typography-Focused Hero**: A clean, centered layout with a dynamic "Availability" status and smooth entrance animations.
- **Interactive Tech Summary**: At-a-glance overview of core services (React, Next.js, Three.js, AI/ML) with grayscale-to-color hover interactions.
- **Animated Testimonials**: A custom, auto-rotating carousel powered by **Framer Motion**, featuring "Voices of Trust" from a professional network with integrated GitHub links for social proof.
- **Balanced Grid Layout**: Standardized 1100px container alignment across all sections (Skills, Work, FAQ, CTA) for a consistent and professional visual flow.
- **Premium Portrait CTA**: A personalized Call-To-Action section featuring a side-by-side layout with high-quality portrait integration.
- **Responsive Design**: Fully optimized for all devices, from large monitors to mobile screens.
- **Performance Optimized**: Built using Next.js App Router for superior loading speeds and SEO.

## 🚀 Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
- **Library**: [React 19](https://react.dev/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/)
- **Styling**: [CSS Modules](https://github.com/css-modules/css-modules) (Vanilla CSS)
- **Icons**: Lucide React & Custom SVGs
- **Deployment**: [Vercel](https://vercel.com/)

## 🛠️ Getting Started

### Prerequisites

- Node.js 18.x or later
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Chiki-tella/exquisite_vibes_only.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## 📂 Project Structure

- `src/components/`: Modular UI components (Hero, CTA, Testimonials, etc.)
- `src/app/`: Next.js App Router pages and global configurations.
- `public/`: Static assets including optimized portraits and work previews.
- `src/styles/`: Global CSS variables and design tokens.
